会議AIの使用方法

            
那須ミッドシティホテル専用AIを最大限に活用するためのガイド

          

          ユーザーマニュアル
        


        

          

            
 1. 会議同席AIの活用

            
    
              
  • モード選択: 議題（AI検索、GBP、料金戦略など）に合わせてモードを切り替えます。
    • 
              
  • 相談内容: 具体的な問いを入力します。「今週の会議論点を作る」ボタンなどの活用も有効です。
    • 
              
  • 会議補足メモ: 今月の稼働データや、役員の懸念事項を貼り付けると回答精度が劇的に向上します。
    • 
              
  • 最新Web調査: チェックを入れると、最新の競合動向やトレンドを含めた回答を行います。
    • 
            

          


          

            
 2. 料金シミュレーター

            
    
              
  • 数値入力: 稼働率やADR、予約ペース指数などの現在の状況を入力します。
    • 
              
  • 推奨価格を試算: ボタンを押すと、RevPAR（販売可能客室1室あたりの売上）を最大化するための改定案をAIが算出します。
    • 
              
  • 判断の参考に: 提示される「判断理由」を確認し、最終的な料金設定の叩き台として活用してください。
    • 
            

          


          

            
 3. AI検索・Google対策

            
    
              
  • サイト観測: 現在の公式サイトがAI検索（Gemini等）にどう認識されているかを確認します。
    • 
              
  • 実行管理: チェックリストに従い、「Googleビジネスプロフィール」や「構造化データ」の更新を管理します。
    • 
              
  • 優先ソース: AIが学習源としている主要サイトの更新状況をチェックします。
    • 
            

          


          

            
 4. 会議メモと保存

            
    
              
  • 決定事項の記録: 会議中に決まったことや次回の宿題を即座にメモします。
    • 
              
  • 保存機能: 「保存」ボタンを押すと、ブラウザを閉じても内容は保持されます（LocalStorageを使用）。
    • 
              
  • データの消去: 会議終了後や情報をリセットしたい場合は「消去」でクリアできます。
